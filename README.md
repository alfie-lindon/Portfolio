# My Portfolio

Personal portfolio website built with Vue 3 + Vite + Tailwind CSS.

## Tech Stack
- Vue 3
- Vite
- Tailwind CSS
- Floating Vue (tooltips)
- Vue PDF Embed

## Project Setup
npm install

## Run Dev Server
npm run dev

## Build for Production
npm run build
